How the boardwalk is built

  • Every environment is Terraform end-to-end — one command up, one command down.
  • Free-tier-first: the always-on set idles at roughly a dollar a month, with hard budget tripwires.
  • Security by default: private buckets behind CloudFront, least-privilege IAM per function, strict headers.
  • Live demos self-heal: seeded data resets nightly, so the door is always open.
