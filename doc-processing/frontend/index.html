What this plank proves

  • Document automation end to end: S3 event → EventBridge → Step Functions orchestration with per-stage failure handling — every document carries its own step-by-step timeline as evidence.
  • OCR & form intelligence: Textract asynchronous FORMS analysis turns scans into text plus labeled key/value pairs with confidence scores.
  • NLP & governance: Comprehend entity extraction feeds the facet index; a PII detector flags documents that would need handling care in a real records system.
  • AI classification: Claude Haiku on Bedrock assigns a document type, title, summary, and primary date — the metadata a records clerk would otherwise key in by hand.
  • Cost containment in depth: Cognito JWT gate (no anonymous uploads, no self-signup), size and page caps enforced before OCR starts, per-user daily caps, and a global daily kill switch that bounds worst-case spend even if a credential leaks.

All infrastructure is Terraform in the public aws-boardwalk repo. Back to the Demo Hub.
