Alpenglow Model Workbench

"Which model should we use?" is a procurement question, not a matter of taste. This bench sends one prompt to four foundation models from four vendors, through the single Bedrock Converse API, and lines the answers up with measured latency, token counts, and computed cost. Every run lands in an audit ledger: who ran what, against which models, with which parameters.

CH 01 · Roster

Four channels on the bench

Four models, four vendors, one Converse call. Swapping models on Bedrock is a parameter, not a re-integration. Prices are published on-demand rates per million tokens, embedded so every response can carry its real cost.

CH 02 · Access

Sign in to run comparisons

Every run fans out to real models and costs real tokens, so this demo requires a credential. It is not printed here, and there is no self-signup; Planetek hands it out with proposals and demos. Runs are capped per user and demo-wide per day.

    CH 05 · Evidence

    What this plank proves

    • Model selection as an engineering discipline: the same prompt, the same parameters, four vendors side by side, with latency, tokens, and cost measured rather than estimated. The "cheapest capable model" conversation becomes evidence-based.
    • One API, any model: Bedrock's Converse API makes the fan-out a loop rather than four integrations. Swapping or adding a model is IAM plus a config entry, which is exactly the exit-ramp a government buyer should demand from a GenAI vendor.
    • Auditable by construction: every run is logged with its parameters, token usage, and computed cost. That is the explainable audit trail responsible-AI policies require, demonstrated rather than promised.
    • Spend that cannot run away: a Cognito gate with no self-signup and an unpublished credential, per-user and demo-wide daily run caps in DynamoDB, a hard output-token ceiling, and a 5 rps edge throttle. Idle cost is $0; a worst-case leaked-credential day is a few dollars.
    • Grounding differences made visible: the refusal-test scenario shows which models admit "the excerpt doesn't say" and which ones confidently invent. For civic AI that is the single most important behavior.
