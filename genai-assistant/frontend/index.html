Answers come only from the demo corpus and include citations. Off-corpus questions are declined by design. Conversations are logged (7-day retention) as the audit-trail half of the responsible-AI pattern.

What this plank proves

  • RAG done cheaply: Titan v2 embeddings + in-Lambda cosine retrieval over an S3 vector index — no vector database, $0 idle.
  • Grounded answers: Claude Haiku 4.5 on Amazon Bedrock, constrained to cited passages with prompt guardrails and a confidence score from retrieval similarity.
  • Cost containment in depth: Cognito JWT gate (no anonymous access, no self-signup), API-edge throttling, per-user daily caps, and a global daily kill switch that bounds worst-case spend even if a credential leaks.
  • Swappable corpus: point it at any domain's documents — building codes, HR policy, grant guidelines — with one sync + re-index command.
  • Audit & feedback loop: every exchange logged with token counts and latency; thumbs up/down feedback captured for review.
